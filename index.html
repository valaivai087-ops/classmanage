Scenario Response Guide · Systems Thinking Approach
⏰
面對遲到
Handling Tardiness
▶
❌ 普通做法
遲到 → 責罵處罰
Late → Scold & punish
✅ 建模做法
遲到 → 找根因 → 修系統
Late → Find root cause → Fix system
1
統計遲到原因 — 用數據代替猜測
Data-based root cause analysis
手機65% / 睡過頭15% / 交通10% / 其他10%
2
睡眠調查 — 找高風險學生
Sleep survey to find at-risk students
少於6小時連續3天 → 重點關懷
3
手機使用調查 — 找熬夜根源
Screen time audit for late-night habits
手機 → 睡眠 → 專注力 → 成績 → 遲到
4
晨間簽到制度 + 早到積分
Early arrival points & tracking system
早到1天+1分；連續一月+30分；可兌換獎品
5
晨間儀式 — 讓學生期待早到
Morning ritual that makes early arrival worth it
感恩→深呼吸→晨讀→英文單字→今日目標
不是讓學生「不敢遲到」，而是讓學生「願意早到」
Not "scared to be late" but "wanting to arrive early"
📱
面對手機問題
Handling Phone Addiction
▶
❌ 普通做法
手機 → 沒收
Phone → Confiscate
✅ 建模做法
手機成癮 → 注意力失衡 → 未來感不足 → 修系統
Addiction → Attention loss → No future vision → Fix system
1
手機停車場 — 先管理環境，不考驗意志力
Phone parking lot — design environment, not willpower
進教室→手機放指定位置→下課領回
2
專注挑戰賽 — 把專注變成遊戲
Focus challenge — gamify concentration
25分鐘專注→完成任務→獲得積分→月表揚
3
每日使用紀錄 — 量化才能優化
Daily screen time log — measure to improve
平均5小時→目標4小時→目標3小時
4
閱讀替代方案 — 替代勝過禁止
Reading alternatives — replace, don't just ban
漫畫、桌遊、英文小說、AI學習區、討論區
5
未來目標牆 — 提高長期感
Dream wall — build long-term thinking
手機提供立即快樂；目標提供未來快樂
不是讓學生「少用手機」，而是讓學生找到比手機更值得投入的目標
Not "use phone less" but "find something more worthy than the phone"

面對成績差
Handling Poor Academic Performance
▶
❌ 普通做法
成績差 → 不夠努力
Poor grades → "not trying hard enough"
✅ 建模做法
成績差 → 找瓶頸 → 設策略 → 精準改善
Poor grades → Find bottleneck → Design strategy
1
錯題分析 — 考卷是找漏洞的工具
Error analysis — tests reveal learning gaps
英文：單字40% / 閱讀30% / 文法20% / 粗心10%
2
弱科診斷 — 不要平均用力
Weak subject diagnosis — target the bottleneck
補最弱科通常進步最快
3
學習小組 — 異質分組，教別人是最好的學習
Study groups — peer teaching maximizes learning
高+中+低混合；每週共同解題、互相教學
4
學霸配對 — 縮短學習距離
High-achiever pairing — bridge the gap
教方法，不是給答案；每週30分鐘陪跑
5
個人改善計畫 — 因材施教
Personal improvement plan — personalized paths
學生A英文弱→每日20字+每日閱讀1篇+每週模考
